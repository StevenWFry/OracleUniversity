# OracleUniversity

Markdown-first Oracle study notes, course breakdowns, labs, and workshop
summaries built from Oracle University material and then cleaned up so a human
can actually use them.

This repo is not a software project pretending to be documentation.
It is a documentation project with occasional pockets of scripts, labs, and
scratch material.

## Folder Conventions

Common patterns in this repo:

- `00_TABLE_OF_CONTENTS.md` for folder-level navigation
- `01_Course_Overview.md` or `course-overview.md` as a course entry page
- numbered lesson files such as `07_Using_Recovery_Manager.md`
- `labs/` folders for hands-on exercises
- occasional supplemental files like exam topics, scratch notes, or summaries

## Reading Strategy

If you are browsing on GitHub:

1. start with the folder-level table of contents or course overview
2. follow the numbered lesson sequence
3. use lab and summary files as reinforcement, not as the main narrative

If you are studying for an Oracle exam:

- use the numbered lessons for structured review
- use the labs and exam-topic files for recall practice
- treat scratch material as helpful, but not canonical

## Caveat Emptor

These notes are intentionally more readable and more sarcastic than the source
material. The goal is retention, not solemnity.

Oracle syntax, parameter names, views, and operational behavior are preserved as
carefully as possible, but this repo is still a study-notes repo, not official
product documentation.
