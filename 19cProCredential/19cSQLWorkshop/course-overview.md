# Oracle Database 19c: SQL Workshop - Course Overview

This folder covers the core Oracle SQL workshop sequence: retrieving data,
filtering and sorting it, joining tables, writing subqueries, handling DML and
DDL, and dealing with the kind of schema objects Oracle insists you learn by
name and by scar tissue.

## What This Workshop Covers

- `SELECT` statements and result-set shaping
- filtering, sorting, and set operators
- single-row and group functions
- joins and subqueries
- DML and DDL in Oracle and MySQL comparison sections
- data dictionary views
- sequences, synonyms, indexes, and views
- schema-object management
- subquery-based data manipulation
- user access control
- advanced queries and time zone handling

## Prerequisites

- basic database concepts
- comfort with tables, rows, and columns
- willingness to let Oracle's naming conventions happen to you
